# OperatorKTV - Система управления медиафайлами

Программа для сетевого контроля воспроизводимых медиафайлов с автоматическим планированием.

**Целевая ОС с GUI:** Windows  
**Подконтрольная ОС:** Linux Lubuntu (Ubuntu 20.04+)

## Описание

OperatorKTV - это система удалённого управления воспроизведением медиафайлов, состоящая из двух компонентов:

- **Windows GUI клиент** - графический интерфейс для управления расписанием и плейлистами
- **Linux daemon** - фоновая служба на удалённой системе для воспроизведения медиафайлов

## Возможности

### Управление расписанием
- Добавление видеофайлов на конкретное время (месяц, день, час, минута)
- Просмотр запланированных воспроизведений в виде дерева
- Включение/отключение запланированных файлов
- Удаление файлов из расписания и с устройства
- Drag-and-drop из Проводника или Total Commander

### Плейлисты
- Создание и управление плейлистами для фонового воспроизведения
- Непрерывное воспроизведение когда нет запланированных файлов
- Автоматическое переключение между расписанием и плейлистами
- Приоритет запланированных файлов над плейлистами

### Временной контроль
- Автоматическое включение вещания в 6:00
- Автоматическое отключение в 22:00
- Настраиваемые часы работы

### Сетевой функционал
- Подключение по SSH/SFTP
- Автоматическая установка ПО на удалённую систему
- Проверка совместимости системы
- Передача файлов с индикацией прогресса
- Встроенная SSH консоль для прямого управления

## Установка

### 1. Установка зависимостей Windows

```bash
pip install -r requirements_windows.txt
```

### 2. Сборка offline пакета для Linux

```bash
python build_offline_package.py --arch x86_64
```

Это создаст `offline_package/ktv_offline_package.tar.gz` с всеми необходимыми компонентами для установки на Linux систему без доступа к интернету.

### 3. Запуск Windows клиента

```bash
python operator_ktv/main.py
```

### 4. Подключение к Linux системе

1. В меню выберите **Файл → Подключиться**
2. Введите IP адрес, логин и пароль Linux системы
3. Если daemon не установлен, программа предложит установить его автоматически
4. При установке будут:
   - Переданы все необходимые файлы
   - Установлен MPV и зависимости
   - Настроен systemd сервис
   - Созданы рабочие директории

## Использование

### Добавление фильма в расписание

1. В вкладке **Фильмы** найдите нужный месяц и день
2. Перетащите видеофайл на день
3. В диалоге выберите время воспроизведения
4. Нажмите **Подтвердить**

Файл будет загружен на удалённую систему и добавлен в расписание.

### Управление плейлистами

1. В вкладке **Клипы** нажмите **Создать плейлист**
2. Введите название плейлиста
3. Перетащите видеофайлы в область справа
4. Дважды кликните по плейлисту для активации

### SSH консоль

Для прямого доступа к Linux системе:
1. **Инструменты → SSH Консоль**
2. Введите команды и нажмите Enter

### Проверка статуса

**Инструменты → Статус daemon** - показывает текущее состояние системы

## Настройка

### Linux конфигурация

Файл: `/etc/ktv/config.json`

```json
{
    "api_port": 9999,
    "media_base_path": "/opt/ktv/media",
    "database_path": "/var/lib/ktv/schedule.db",
    "log_path": "/var/log/ktv/daemon.log",
    "broadcast_start": "06:00",
    "broadcast_end": "22:00",
    "mpv_path": "/usr/bin/mpv"
}
```

После изменения конфигурации:
```bash
sudo systemctl restart ktv-daemon
```

## Управление сервисом на Linux

```bash
# Проверка статуса
sudo systemctl status ktv-daemon

# Перезапуск
sudo systemctl restart ktv-daemon

# Просмотр логов
sudo journalctl -u ktv-daemon -f

# Остановка
sudo systemctl stop ktv-daemon

# Запуск
sudo systemctl start ktv-daemon
```

## Устранение неполадок

### Daemon не запускается

1. Проверьте логи: `journalctl -u ktv-daemon -n 50`
2. Проверьте, что MPV установлен: `which mpv`
3. Проверьте права доступа: `ls -la /opt/ktv`

### Не удаётся подключиться

1. Проверьте SSH доступ: `ssh user@ip`
2. Проверьте брандмауэр на Linux
3. Проверьте, что daemon запущен

### Файлы не воспроизводятся

1. Проверьте формат файла (должен поддерживаться MPV)
2. Проверьте расписание: в GUI или через SSH консоль
3. Проверьте временной контроль (6:00-22:00)
